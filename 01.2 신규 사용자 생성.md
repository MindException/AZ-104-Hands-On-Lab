# 1.2 신규 사용자 생성
![](./MD_Images/1-2_001.jpg)

## 1. 신규 사용자 생성

### 1. 신규 사용자 만들기
![](./MD_Images/1-2_002.jpg)
* `Microsoft Entra ID` 에서 사용자 탭으로 이동한 후, 새 사용자 만들기를 클릭합니다.

<br>

### 2. 사용자 기본사항 설정
![](./MD_Images/1-2_003.jpg)
* 사용자 ID/PW와 표시되는 이름을 설정합니다.   


<br>

### 3. 사용자 세부 속성 설정
![](./MD_Images/1-2_004.jpg)
* 사용자의 부서 혹은 직함 등을 추가로 설정합니다. 

<br>

### 4. 새 사용자 생성
![](./MD_Images/1-2_005.jpg)
* 설정한 내용을 검토한 후, 신규 사용자를 생성합니다.  

<br>

### 5. 생성된 사용자 확인
![](./MD_Images/1-2_006.jpg)
* 목록을 새로고침하여 생성된 사용자를 확인합니다.  

<br>

## 2. 외부 Guest 사용자 생성

### 1. 외부 사용자 초대
![](./MD_Images/1-2_007.jpg)
* `Microsoft Entra ID` 에서 사용자 탭으로 이동한 후, `외부 사용자 초대`를 클릭합니다.

<br>

### 2. 외부 사용자 초대 설정
![](./MD_Images/1-2_008.jpg)
* 초대할 외부 사용자의 이메일 정보 및 추가 정보를 입력합니다.

<br>

### 3. 외부 사용자 세부 속성 설정
![](./MD_Images/1-2_009.jpg)
* 초대할 외부 사용자 계정의 직함 및 부서 정보 등을 추가합니다.  

<br>

### 4. 외부 사용자 초대
![](./MD_Images/1-2_010.jpg)

<br>

### 5. 외부 사용자 계정 생성
![](./MD_Images/1-2_011.jpg)
* 외부 사용자의 수락 여부와 상관 없이 바로 계정이 생성됩니다.  

<br>

### 6. 이메일 수락
![](./MD_Images/1-2_012.jpg)
* 위와 같이 Entra 가입 이메일을 수신받게 됩니다.
